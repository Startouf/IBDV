Custom data visualisation

Let it rock :D

Input

  • 2 dimensional data, and 1+n param per data object : [ Description, Number Views, Number of comments, Number vote Up, Number vote Down, etc.]
  • Not yet sure about the params that could go in n : see below

Idea behind it :

I'm going to work on a collaborative wiki this summer, and for example, people are going to post their ideas on some points, and it could be cool to have a visualisation of all their ideas, and their "hotteness" i.e. if many people posted comments, replies, etc. So, I'd like to get a global view of all the ideas, sort them by hotteness, and draw some kind of scatterplot.

How to visualise

Ideas will be sorted by thei "hotteness", and the first one will be displayed in the center of the screen as a big "bubble". Then, the first other ones (by hotteness) will be on a circle around the first idea (for example, a circle that can contain 9 bubbles Then, the second batch will be displayed on a bigger circle, that can contain more ideas Etc. Till there are no more ideas to place

The bubbles can have different properties to represent the n parameters :
The size : could represent the global hotteness (like a ||.||2 of all the values)
The color : could represent the overall approbation of the idea (number Votes Up/number Votes Down)
What else ?

Current data format

A 2D Array [[data1],[data2],...] where each data is an array of :

  • String : title of idea
  • Number of comments
  • Number of Votes Up
  • Number of Votes Down
